Block 2: Weeks 6–9 — Volume Ramp

Add a sixth set to main compound lifts. Rep targets tighten on upper/lower strength days. Accessories stay similar but load increases. This is where most of the hypertrophy work happens. RPE climbs week by week.

Target RPE: 8–9 on compound working sets.
Nutrition: Hold 200–400 kcal surplus. Bodyweight should be trending up 0.3–0.5 kg/week.
Auto-regulation: If a weight feels heavier than expected, keep it — don’t drop the load. If a set goes past RPE 9, end the set.
