Block 3 — Peak Volume · Weeks 9–11

Highest volume block. This is where growth happens. RPE 8–9. Additional sets on all exercises. Intensity techniques introduced in Week 11.

This block is hard. Sleep 7+ hours. Eat in a surplus. If you're not recovering between sessions, reduce volume by 1 set per exercise rather than pushing through.

Key Changes from Block 2

  • +1 set on most exercises compared to Block 2 Week 7
  • Week 11 introduces intensity techniques on final sets: drop sets on machine work, myo-reps on cables, partials on isolation
  • RPE 8–9 across the board — you should be close to failure on most working sets
  • Total weekly sets per muscle group: 22–24

Week 11 Intensity Techniques

TechniqueHow to ApplyWhen
Drop SetAfter final set, reduce weight 25–30% and go to failure. One drop only.Machine exercises (chest press, leg press, leg ext)
Myo-RepsHit ~12 reps to failure, rest 5 breaths, do 3–5 reps, repeat 3 times.Cable exercises (flys, laterals, curls)
PartialsAfter failure on full reps, continue with half-reps in the shortened range.Lateral raises, curls, extensions
Lengthened PartialsAfter failure, do 4–5 reps in the bottom stretched position only.Incline curls, flys, pullovers
