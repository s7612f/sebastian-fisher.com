## Overview
This repository contains the production-ready static website for **Sebastian Fisher**, an elite personal elevation concierge delivering six-to-ten week immersions for discerning professionals. The site is lightweight, responsive, and deployable directly to GitHub Pages.

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sebastian-fisher.com.git
   ```
2. Open `index.html` in your preferred browser, or
3. Enable GitHub Pages for the repository to host it publicly.

## Form Handling
The inquiry form currently posts to a placeholder Formspree endpoint at `https://formspree.io/f/YOUR_ID`. Replace `YOUR_ID` with your Formspree form identifier before going live.

## Technologies
- HTML5 with semantic structure and ARIA support
- CSS3 with custom properties, Flexbox, and CSS Grid
- Vanilla JavaScript for interactive behaviors and accessibility enhancements

## Performance & Accessibility
- Mobile-first responsive design with breakpoints at 320px, 768px, and 1024px
- WCAG 2.1 AA considerations including proper landmarks, focus states, and readable contrast
- Lazy-loaded imagery and minimal assets to keep the total footprint under 2MB
